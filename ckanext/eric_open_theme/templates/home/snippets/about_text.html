The Eawag Research Data Institutional Repository (ERIC)

This is the new release of Eawag's research data repository. It features a new meta-data schema that is better adapted to user's needs, improved workflow support, new functionality such as restricted resources, usability improvements and bug fixes.

Terms of Use:

  1. Use of data in this repository only with permission from Author or Usage Contact!
  2. Making available data in this repository outside Eawag only with permission from Author or Usage Contact!

Underlying software

  • The CKAN has been upgraded from version 2.5.2 to version 2.7.3 .
  • The search platform SOLR has been upgraded from version 5.3.1 to version 6.5.0 .
  • The database Postgres has been upgraded from version 9.4.15 to version 9.6.9 .

Server

Instead of on a single virtual machine (VM), Postgres and SOLR now run on a separate server. The storage backend is now connected to the new storage system at Eawag which provides a total storage capacity of 11 TB for ERIC.
